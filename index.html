Project: Smart AQI Monitoring System Using Web API

A low-cost, web-based real-time air quality monitoring system designed to provide essential health advisory and pollution awareness to users.

Key Objectives

  • Real-time Air Quality Index monitoring
  • Health advisory based on pollution levels
  • Pollution awareness and impact assessment
  • Smart alerts for hazardous conditions

Features

  • Live AQI display with color-coded indicators
  • Personalized health warnings and precautions
  • Pollution impact calculation (cigarette equivalent)
  • Historical trend analysis and visualization
  • AI-powered AQI predictions
  • Detailed pollutant breakdown (PM2.5, PM10, O₃, NO₂, SO₂, CO)
